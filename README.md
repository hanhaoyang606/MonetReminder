# MonetReminder 🎨

**MonetReminder** is a modern, elegant Android reminder application built with **Kotlin** and **Jetpack Compose**. It fully embraces the **Material You (Monet)** design language, providing a personalized experience with dynamic color theming that adapts to your wallpaper.

**MonetReminder** 是一款现代、优雅的 Android 提醒应用，基于 **Kotlin** 和 **Jetpack Compose** 构建。它全面拥抱 **Material You (Monet)** 设计语言，通过适配壁纸的动态配色方案，为您提供极具个性化的使用体验。

---

## ✨ Features / 功能特性

- **🎨 Material You Design**: Beautiful dynamic color support on Android 12+. (在 Android 12+ 设备上支持精美的动态色彩)
- **🚀 Jetpack Compose**: Built with modern declarative UI components for a smooth experience. (使用现代声明式 UI 组件构建，确保流畅的交互体验)
- **⏰ Smart Reminders**: Efficient local reminder management using `AlarmManager`. (使用 `AlarmManager` 实现高效的本地提醒管理)
- **📱 Clean Architecture**: High-quality, maintainable Kotlin codebase. (基于高质量、易维护的 Kotlin 代码库开发)

## 🛠️ Tech Stack / 技术栈

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Design System**: Material Design 3 (Material You)
- **Architecture**: MVVM / Clean Architecture

## 🚀 Getting Started / 快速开始

1. **Clone the repository / 克隆仓库**:
   ```bash
   git clone https://github.com/hanhaoyang606/MonetReminder.git
   ```
2. **Open in Android Studio / 在 Android Studio 中打开**:
   Use Android Studio Hedgehog (or newer) for the best experience. (建议使用 Android Studio Hedgehog 或更高版本)
3. **Build and Run / 编译并运行**:
   Sync project with Gradle files and run on your device or emulator. (同步 Gradle 文件，并在您的设备或模拟器上运行)
