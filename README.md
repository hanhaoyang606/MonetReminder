# MonetReminder 🎨

[English](./README.md) | [简体中文](./README_zh.md)

**MonetReminder** is a modern, elegant Android reminder application built with **Kotlin** and **Jetpack Compose**. It fully embraces the **Material You (Monet)** design language, providing a personalized experience with dynamic color theming that adapts to your wallpaper.

## ✨ Features

- **🎨 Material You Design**: Beautiful dynamic color support on Android 12+.
- **🚀 Jetpack Compose**: Built with modern declarative UI components for a smooth experience.
- **⏰ Smart Reminders**: Efficient local reminder management using `AlarmManager`.
- **📱 Clean Architecture**: High-quality, maintainable Kotlin codebase.

## 🛠️ Tech Stack

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Design System**: Material Design 3 (Material You)
- **Architecture**: MVVM / Clean Architecture

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/hanhaoyang606/MonetReminder.git
   ```
2. **Open in Android Studio**:
   Use Android Studio Hedgehog (or newer) for the best experience.
3. **Build and Run**:
   Sync project with Gradle files and run on your device or emulator (Android 12+ recommended for dynamic colors).
