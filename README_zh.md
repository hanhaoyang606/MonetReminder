# MonetReminder 🎨

[English](./README.md) | [简体中文](./README_zh.md)

**MonetReminder** 是一款现代、优雅的 Android 提醒应用，基于 **Kotlin** 和 **Jetpack Compose** 构建。它全面拥抱 **Material You (Monet)** 设计语言，通过适配壁纸的动态配色方案，为您提供极具个性化的使用体验。

## ✨ 功能特性

- **🎨 Material You 设计**: 在 Android 12+ 设备上支持精美的动态色彩。
- **🚀 Jetpack Compose**: 使用现代声明式 UI 组件构建，确保流畅的交互体验。
- **⏰ 智能提醒**: 使用 `AlarmManager` 实现高效的本地提醒管理。
- **📱 简洁架构**: 基于高质量、易维护的 Kotlin 代码库开发。

## 🛠️ 技术栈

- **开发语言**: Kotlin
- **UI 框架**: Jetpack Compose
- **设计规范**: Material Design 3 (Material You)
- **项目架构**: MVVM / Clean Architecture

## 🚀 快速开始

1. **克隆仓库**:
   ```bash
   git clone https://github.com/hanhaoyang606/MonetReminder.git
   ```
2. **在 Android Studio 中打开**:
   建议使用 Android Studio Hedgehog 或更高版本以获得最佳开发体验。
3. **编译并运行**:
   同步 Gradle 文件，并在您的设备或模拟器上运行（推荐 Android 12+ 以支持动态色彩）。
